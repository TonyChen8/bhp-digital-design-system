Popover

Popovers are designed to give secondary content about a particular element to a page or contain further options/actions for the user in a container. The popover interaction is more efficient for small interactions because it keeps the user in the context of the action.

When and how to use this component:

  • When you have more information you need to present to the user as a secondary.
  • When you have too much information to fit in a Tooltip*.
  • Only one popover can be open at any one time.
  • Popovers can be fired from any clickable object within an interface including buttons, clickable icons, and links.
  • Clicking anywhere on the screen should dismiss the popover.
  • If a screen is resized, the popover should open witn the viewport at all times.
info If you require more detailed secondary information, consider using a modal dialog instead.
  • React

Usage

The state of the popover is controlled using `isOpen`.

Example
import React from 'react';
import { Button, Popover, PopoverHeader, PopoverBody } from 'reactstrap';

export default class Example extends React.Component {
  constructor(props) {
    super(props);

    this.toggle = this.toggle.bind(this);
    this.state = {
      popoverOpen: false
    };
  }

  toggle() {
    this.setState({
      popoverOpen: !this.state.popoverOpen
    });
  }

  render() {
    return (
      <div>
        <Button id="Popover1" onClick={this.toggle}>
          Popover on right
        </Button>
        <Popover
          placement="right"
          isOpen={this.state.popoverOpen}
          target="Popover1"
          toggle={this.toggle}
        >
          <PopoverHeader>Popover Title</PopoverHeader>
          <PopoverBody>
            Lorem ipsum dolor sit amet, conse ctetur adipisicing elit,
            sed do eiusmod tempor incididunt ut laborerem ipsum dolor sit
            amet, conse ctetur adipisicing elit, sed do eiusmod tempor
            incididunt ut labore.
          </PopoverBody>
        </Popover>
      </div>
    );
  }
}

Positioning

The position of the popover can be controlled with the `placement` property.

Example



import React from 'react';
import { Button, Popover, PopoverHeader, PopoverBody } from 'reactstrap';

class PopoverExample extends React.Component {
  constructor(props) {
    super(props);

    this.toggle = this.toggle.bind(this);
    this.state = {
      popoverOpen: false
    };
  }

  toggle() {
    this.setState({
      popoverOpen: !this.state.popoverOpen
    });
  }

  render() {
    const { buttonText, placement, id, withTitle } = this.props;
    return (
      <div>
        <Button id={id} onClick={this.toggle}>
          {buttonText}
        </Button>
        <Popover
          placement={placement}
          isOpen={this.state.popoverOpen}
          target={id}
          toggle={this.toggle}
        >
          {withTitle &&
            <PopoverHeader>Popover Title</PopoverHeader>
          }
          <PopoverBody>
            Lorem ipsum dolor sit amet, conse ctetur adipisicing elit,
            sed do eiusmod tempor incididunt ut laborerem ipsum dolor sit
            amet, conse ctetur adipisicing elit, sed do eiusmod tempor
            incididunt ut labore.
          </PopoverBody>
        </Popover>
      </div>
    );
  }
}

export default () => {
  return (
    <div>
      <PopoverExample
        id='PopoverLeft'
        buttonText='Popover on left with title'
        placement='left'
        withTitle
      /><br/>

      <PopoverExample
        id='PopoverLeftNoTitle'
        buttonText='Popover on left without title'
        placement='left'
      /><br />

      <PopoverExample
        id='PopoverTop'
        buttonText='Popover on top'
        placement='top'
      /><br />

      <PopoverExample
        id='PopoverBottom'
        buttonText='Popover below'
        placement='bottom'
      />
    </div>
  );
}
